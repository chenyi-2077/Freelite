## 写作要求

| 项目 | 要求 |
|---|---|
| **语言** | 中文 |
| **格式** | Markdown（`.md` 文件） |
| **存放位置** | `Freelite/docs/说明书_模块名.md` |
| **字数** | 每份 2000-4000 字 |
| **协作方式** | 每人写完后 commit & PR 到主仓，组长 review 合并 |

---

## 时间节点

| 阶段 | 时间 | 说明 |
|---|---|---|
| 初稿完成 | T + 3 天 | 每人按分工完成自己的说明书 |
| 组长 review | T + 4 天 | 组长审核四份 |
| 整合版完成 | T + 5 天 | 组长完成整合版 |
| 终稿提交 | T + 6 天 | 全部定稿 |
