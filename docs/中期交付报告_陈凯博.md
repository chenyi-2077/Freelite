## 一、项目简介

Freelite 是一个 B/S 架构的 Web 应用，对标 Freelancer.com 的核心功能，技术栈选用 **JSP + Bootstrap 5 + Java Servlet + JDBC + MySQL 5.7 + Tomcat 8.5**。

目前已完成的核心业务链路：

```
注册账号 → 登录系统 → 浏览项目 → 发布项目 → 投递竞标 → 查看竞标 → 查看订单 → 评价
```

## 三、项目文件结构

│   ├── dao/                         ← 数据访问层（6个）

│   └── util/                        ← 工具类（2个）

## 四、核心功能演示场景

### 场景：从注册到评价的完整流程

```
① 注册账号（陈怡安）
    └── 选择"雇主"角色 → 填写邮箱密码 → 创建成功自动登录
    
② 发布项目（陈凯博）
    └── 填写标题"开发一个博客系统" → 预算5000 → 选择Web开发分类 → 发布成功
    
③ 浏览项目（陈凯博）
    └── 项目市场列表展示 → 可按关键词搜索 → 可按分类筛选 → 分页浏览
    
④ 投递竞标（陈僖睿）
    └── 自由职业者账号登录 → 点击项目 → 填写报价3000+15天 → 提交竞标
    
⑤ 查看竞标（陈僖睿）
    └── 访问竞标列表页面 → 看到所有竞标信息（金额/工期/状态）
    
⑥ 查看订单（陈子豪）
    └── 订单列表页 → 显示所有参与订单及状态标签
    
⑦ 提交评价（陈子豪）
    └── 对合作方评分1-5星 → 填写评语 → 提交评价
```

## 五、运行方式

1. 执行 `docs/database.sql` 创建数据库和表
2. 修改 `DBUtil.java` 中的数据库连接信息
3. 导入 Eclipse → Run on Server（Tomcat 8.5）
4. 测试账号：`alice@test.com / 123456`（雇主）、`bob@test.com / 123456`（自由职业者）
